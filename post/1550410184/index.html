介绍

Baidu-PCS-Go 是一款支持 PC/移动设备 的仿 Linux shell 文件处理命令的百度网盘命令行客户端.

使用（必看！！！）

进入终端输入 cd /baidu/ && ./BaiduPCS-Go 即可运行
将想下载的文件移动到 /我的应用数据/baidu_shurufa/
接着执行 cd /app/baidu_shurufa/
然后输入 d 将下载的文件名字 即可下载

一些命令&操作技巧
help 帮助
ls 可列出当前文件和文件夹
cd 路径 可进入某路径
d 你要下载的文件 下载某文件
locate 你要获取直链的文件 获取某文件直链
logout 退出登录
停止下载按住 ctrl 并键入 c 即可
多任务下载直接多开窗口即可
