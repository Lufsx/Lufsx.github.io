TL;DR
想着入手 Pixel 2 有些时日了，便写了篇体验文总结下。
注：本文全程唠嗑，纯属个人主观，扒扒 Pixel 2 有的没的细节 😋

Google 正式结束对 Pixel 2 系列更新支持。

啊，转眼就到了 Android 11 正式版了，已经快要停止更新支持了，这篇文章也只是字词上的小修小补，并没有说最近的变化 🤣

本身的文字功底也不好，最近也有推倒重来的想法，感谢 搞机日记 指出的部分细节以及让我有推倒重来的想法

缘起

去年 11 月，在被种草长达一年后，在将 米 6 卖给同学后，终于可以将 Pixel 2 收入囊中啦 😆

中间有个小插曲，我星期五在某宝下单蓝色后，以为星期日能到，结果店家打电话过来说蓝色的要多等，于是转手换了个黑色的立马发货
现在看看，黑色的 Pixel 2 真的是最没有质感的 🤣

外观

外观嘛。。。总体「大黑边 + 大额头 + 拼色」设计

除大号独占的熊猫色外，颜色共有三种，分别为“Kinda Blue（有点蓝）”、“Just Black（就是黑）”和“Clearly White（纯纯白）”（黑色是最没有质感的！！！

正面上下有对称双扬，仔细看的话还能在正面看到七个天线溢出点，配上 5.0 英寸的 AMOLED 屏幕，屏占比尤其的低。

电源键被放置在机身上半部，音量键也在机身中部偏上的位置，不常规，但习惯后还好。

背面则有一颗 永流传的 IMX 363 362，还有常规的后置指纹，分别置于玻璃与金属两侧。

玻璃部分光泽极佳，且反光集中，质感完美。金属部分有一层塑料的「高级涂层」，打造出一种类似于磨砂的质感，拿着很是舒服 😋（当然，也有人吐槽太像塑料不好看



这种「特立独行」设计，喜欢的人自然爱不释手，不喜欢的人怎么安利也提不起劲儿。

而我，就是那种喜欢这种感觉的人，个人又偏爱 Android，这就让我对 Pixel 2 的好感蹭蹭上升～（大黑边除外

当然，信仰嘛，加分的还有背后的大 G 呢

硬件

2017 年的旗舰，当然要用安卓旗舰统一发配的 Snapdragon 835。在经历了 炎龙 810、火龙 82x 后，众人对 835 这一性能与功耗均衡的 SoC 大呼真香，即使在 2020 年，也还算不拖垮。

Pixel 2 有 4+64/128 可选（为 Samsung MLC，UFS 2.1），但那时候，国内一众机器都有「激进」的 6G RAM，Pixel 2 等国外「洋垃圾」都还是 4G RAM

CMOS 是使用的 IMX362，是那 12.2 MP 1.4 µm Dual pixel 的熟悉参数，殊不知与这颗相同规格的 CMOS 将会一直被 Pixel 系列沿用至 2020 年~

谷歌还与牙膏厂定制了一颗“Pixel Visual Core”芯片用来加速处理 HDR+，这使得处理速度远远大于其他手机～

当然，这颗“Pixel Visual Core”也不是什么省油的灯，当你打开相机时，这颗芯片便会立即运作起来，听闻在处理时功耗会瞬间达到恐怖 6W，而处理完毕后也会瞬间关闭。

小号屏幕是使用的是 Samsung FHD AMOLED，高达 441 的 PPI（AMOLED 等效 360），由于是 5.0 英寸，屏幕的细腻程度还是不错的。这块屏幕似乎比大号 LG QHD P-OLED 好了不少，手上这部屏幕稍有老化，观感偏黄 🤣

正面大额头大下巴，不过为了 正面对称双扬 忍了（反复强调

为了实现“Active Edge”，Google 还在机身内塞下了长长的压力传感器，日常捏一捏还能锻炼下身体呢（虽然这个功能只是为了唤醒 谷歌助理 和 静音。。。

更惊喜的是，Pixel 2 系列配备了一颗调教挺不错的「横向线性马达」，在使用 Gboard 时，在拖动光标时，会给人哒哒哒的快感～

但美中不足的是，小号电池只有可怜的 2700mAh，加上电池损耗，续航尿崩。。。

好在充电方面支持 USB Power Delivery 快速充电（不支持 QC 快充），网上有消息说支持 27W（即 9V3A），个人实测冲入功率也能最高达到 22W

还有这个 Type-C 是 USB 3.1 Gen 1（支持 USB 3.0）（USB-IF 窒息命名（手上也没有 3.0 的线，无法测试，实际速率我就不知道啦，我只知道 2.0 下稳定 30MB/s）(确定支持 USB 3.0)

Type-C 转 3.5 的转接线必须带有 DAC 才能在 Pixel 上正常使用

再顺带一提，Pixel 2 还支持 eSIM，不过与我就没啥关系了

相机

Pixel 2 发售时在 DXO 以 98 分的成绩斩获榜一，这便是我第一次听见“Pixel”这个名号，直至今天，那份拍照惊艳依旧存在，不过在如今的旗舰衬托下，显得有那点无力。

相机放到今天来看，我觉得远远不止 99 分，排排我觉得还有个 tier 1，它还拥有在 2020.01.15 之前的谷歌相册无限原图备份（毕竟 Google 的手机可以随软件更新以提高拍摄质量 甚至还可以使用其它开发者移植的谷歌相机来实现一些新机器上特有的功能）

默认是开启 HDR 使用，加持 ZSL(零快门延迟)，可以在不影响普通拍照体验的情况下输出 HDR 照片，用其它手机拍普通 HDR 的体验还可以获得更加暴力的 HDR+ 照片

拍照在 Gcam 上体验绝佳，不论是拍照体验还是成片质量，但在 PVC 的加持下，功耗就。。。

更加蛋痛的是，即使是第三方应用调用摄像头，也会比普通手机消耗更多的电量，这让它原本就尿崩的续航更加不堪（注：PVC 在 Android 8.1 2018.01 更新之后才被启用

大致说下体验，官方相机无法调整参数，且调教相对偏重成片时间，噪点相对较多（除非环境光照实在低的时候才会延长成片时间～

天文模式无缘 Pixel 2，但是可以使用第三方开发者移植的谷歌相机开启天文模式 😋（天文模式是真滴可以拍星星呀

录像方面嘛

防抖有 OIS 与 EIS 双重加持，在当年稳如老狗，放在现在也还不算弱

遗憾录像不支持 4K 60FPS，且在 4K 30FPS 下，果冻效应明显（应该是从左到右扫描～

1080P 则表现极佳，个人建议除非有特殊需求，否则不要开启 EIS，因为画面会进一步裁切，让本来就不大的视野更小

但在光线良好的环境下，有 OIS + EIS 的 1080P 60FPS 真的是稳得不行

光线差的情况下，各模式在对焦速度方面有所损失，1080P 60FPS 相较于 1080P 30FPS 细节大打折扣（你问我 4K 30FPS？噪点大合集！🤣

使用体验

手感&外观

手感嘛。。。143g 的重量与 69.7 的宽度在 2020 砖头遍地的情况下显得格格不入 （大黑边！

握持时，在「高级涂层」的加持下，不滑有质感摸着超舒服的说

外观设计极其特别，亮面玻璃与哑光涂层衔接，一打眼的差距，辨识度挺高的说（然而大部分人都不知道这是什么手机毕竟 404 小公司

系统体验

我觉得系统可以单独拎出来讲，毕竟部分人买它的理由很大一部分是因为它是能最早得到更新的手机

包括我，Google 承诺给 Pixel 2 系列提供长达 3 年的软件更新，现如今已是 2020 年，大限将至 😂（Google 在 2020.10 之后不再保证为 Pixel 2 提供更新

404 Google 的手机，得需要有良好的网络环境才能体验到 原汁原味、轻快流畅 的全家桶（

在 Android 10 中，增加了对手势导航的支持（即「全面屏手势」（Google 为了寻找最佳手势解决方案，经过多方面的研究调查，最终得出了与 iOS 一致的手势方案（小白条～ 🤔）。实际体验不错，返回与侧栏之间也有专门的处理（边缘按住停留再滑动）

系统流畅度绝对是毋庸置疑的，在 Google 的「原生系统」配合下，动画的流畅与轻快完美（尤其是手势动画）体现在各个方面，非线性加上细节，体验特别棒，比 国内各种 UI 都舒服了很多～（这动画我觉得可以摸到 iOS 的说

电信用户若想愉快玩耍，需要进行一些特殊操作才能正常使用电信（此方法据说可以让国内三大家的 VoLTE 正常在 Pixel 2 上使用哦 😝


特殊操作后需要在手动开启 VoLTE 服务哦

有非常多的「手势」设置，包括 Active Edge 在内的共有 10 多种手势，我很喜欢的有「翻转即可设为勿扰」、「快速打开相机」和「滑动指纹即可场刊通知」，这几个都是我日常高频使用的几种手势的说。

特色/不特色功能

「Active Edge」

俗称捏一捏，可以以此途径唤醒谷歌助理，和闹钟等什么响的时候静音

若不想唤醒谷歌助手，则可以在 Magisk 里使用 Active Edge Mof for Pixel devices 配合对应应用即可自定义～（虽然我不用这个，不过还是提一嘴

「闻曲知音」

个人测下来，准确率超高，可以识别大部分欧美音乐，部分 ACG 音乐也能正常识别。之前在学校甚至还识别出了午休铃声呢 😆

「Smart Lock」

其实这个东西基本有比较全的 Play 服务就有，例如 OpenGapps 只要 Nano 以上的版本就有。

个人开启了「贴身检测」，走动时不时拿出手机倒是不用解锁直接打开，蛮方便的说。（但是 Pixel 2 指纹位置那么顺手，真的需要嘛？我也不知道为什么我会开 🤣

但谷歌在 Android 10 上把可信面孔给阉割掉了，理由是 2D 人脸的安全性问题。

「深色主题」

在三月布丁补丁中，Google 加入了随时间自动切换深色主题

个人是一直开着深色主题，毕竟 AMOLED 黑色省电，有适配深色主题并能跟随系统切换的应用便会自动开启深色主题，没适配的可以尝试「开发者选项」中的「覆盖“强制启用 Smart Dark 功能预设”的设置」（别对这个选项抱有太大的期望，按需开启）

Chrome 也支持网页随深色主题自动切换（实测手机端 Microsoft Edge 不支持网页随主题设置切换颜色，顺带一提本博客已支持随主题设置切换深色主题）

性能&游戏体验

性能方面则是我们熟悉的那个高通晓龙 835，Google 使用的是 EAS 内核，相较于一般的 HMP 内核，更加省心，一般来说不需要过多的调教就能获得较好的性能&能效表现

娱乐兔：

总分：255815
CPU：80766
GPU：91594
MEM：41430
UX：42025

个人是刷了 Sultan 内核的，也使用了 YC 大佬的内存优化模块，后台与流畅度都完全不输 19 年旗舰，也想像不出这是部 4G RAM 的手机，下文测试环境均为以上配置 😋

你问我 835 为啥要测？这可是我的主力机啊

近期更新新的测试（

点击显示/隐藏

首先是 绝地求生:刺激战场 和平精英。正式服，画质为 流畅 极限 电影 抗锯齿 阴影 😝



团队竞技，平均 59 FPS。GPU 不动，虽然我知道你吃鸡要高频低载，但你没主界面占用高就过分了啊 🤔

不过配合上下对称双扬，虽然效果不咋滴，但至少俩边音量一样大，打游戏时就不用转接线带个耳机了 🤣

还有这个手机也不需要担心「死亡之握」，这可比 XC 那拿着拿着就断网的好多了 🤣

然后三蹦子。分辨率高，60FPS，其它设置除 HDR 外全关



开放世界全 SSS 或以上的委托，中间帧率为零的是在加载中，大体还行，都是一颗 835 正常的水准（你问我为啥不测后崩坏书？因为我没激活 🤣

Pixel 2 其实也可以玩降压的说，可以自己使用工具自行降压，工具的 GitHub 项目地址

最后说一下 Sultan Kernel，这个内核在 03.05 版本之前，CPU 最低频率为 1.8GHz，之后为了更好的电池能效，将最低 CPU 频率降到 1.4GHz 了（真的狠，但也是真的流畅 🤣（嘛，虽然官方内核就已经调教的很好了

日常使用体验

「原生系统」相对国内各大 UI 倒是缺少了很多本地化内容，不过这些东西或多或少的可以使用第三方应用补齐

个人不太喜欢过多的「本地化」，适度的就差不多了，否则会显得系统极其「臃肿」此处应有三星使用手册，好动画配上轻快流畅简陋的系统，那体验是不可多得的说

续航方面则是 Pixel 2 的硬伤，小号只有 2700mAh 的电池，尤其是像我这样电池损耗成这样的（还当主力打游戏的 😢

外出必带充电宝，不带就完蛋，如果还拍些照片的话，PVC 的功耗可不是闹着玩的。（目前某宝上也没有看到比较持久的电池，唯一的那一家好像电池用不久，有用过的麻烦说下续航情况

其实在这种情况下，轻度使用续航也还算能看（我吹爆 Sultan 内核！

本地化

「原生系统」唯一的本地化可能就是那个来电归属地显示吧

信息里的「垃圾信息防范」开启后也没给我防范一条垃圾短信 🤔

Google 的自动填充可以填写验证码，但似乎国内的验证码压根无法识别出来（个人使用了 Nevolution 搭配插件来实现 验证码复制、填充 等功能的说）

快递进度有启动速度极快的支付宝，长截图可以使用 PPIICC，自动化方面可以使用 Tasker 来到成各种骚操作。

部分功能也可以通过 Riru Magisk 模块开启，例如 Google Feed 可以使用 Riru - Location Report Enabler 模块开启，除此之外，我还使用来存储重定向配套模块的说（Rikka 全家桶 😋

Google Feed 推荐的内容整体上还是不错的，界面也好看，吹爆 😆

Riru 还有个开启支付宝、微信和淘宝的指纹支付模块，需要输入支付密码，以便之后的指纹支付（个人不保证绝对安全，按需下载使用（GitHub 个人主页，点进对应项目的 Release 自行下载）

总结

这是一部 2017 年的旗舰机，影像方面放在 2020 也还有不俗的实力，在预算不够的情况下，也可以将 Pixel 2 当作卡片机，拍照玩挺不错的

优点

这部手机的优点总结下，就是：

  1. Google 信仰
  2. Google 亲儿子，可以最快系统软件更新（即将过期～
  3. 优秀的拍照表现
  4. 上下对称双扬
  5. 有一颗调教很不错的线性马达
  6. 独树一帜的外观设计
  7. 优秀的手感
  8. AMOLED 的纯黑省电
  9. 「原生」系统的流畅轻快
  10. Google Photos 无限原图备份（即将过期
  11. 轻松访问您使用最常用的谷歌应用程序（自带全家桶~）

缺点

缺点总结下，就是：

  1. 上下大额头下巴，宽边框，极低的屏占比
  2. 不堪入目的续航
  3. AMOLED 的 PWM 调光以及老化
  4. 原生系统的轻快流畅「简陋」
  5. 缺失的本地化
  6. 上下对称双扬素质一般
  7. 艰难访问您使用最常用的谷歌应用程序（国内网络环境）

购买建议

喜欢 Google 想要较好的体验又预算不足，直接上 Pixel 2 系列即可（实在不行，上一代 Pixel
喜欢原生系统，想要流畅简洁，更新迅速的无脑 Pixel 系列（注意更新支持年限
想要拍照的卡片机，又限于预算，可以考虑 Pixel 2
有良好的网络环境/海外党，会正确甄别洋垃圾的可以考虑
介意续航的上 XL，介意 XL 屏幕的话就别考虑这一代啦
作为一个备用机还是很不错的（我主力 🤣
其他的话，一律不建议购买
