申请条件

放心，要求不高～

  • 无侵入性广告
    • 国内部分「免费空间」与「免费主机」搭建带有广告 Banner 等 不可控 的广告的站点，不接受申请。
  • 非「免费」域名
    • 建议使用自己购买的顶级域名
    • 此处「免费」域名定义 不包括 以下免费子域名服务：
      • github.io
      • gitlab.io
      • now.sh 等存在于 Public Suffix List 的子域
  • 非采集、无 任何 原创内容的站点
    • 内容不追求多、质量好，但要符合大众道德观念
    • 单独的资源站不接受，抱歉
  • 请注意长期无法访问的站点可能将被移除，若更换域名，请及时更新资料
    • 已移除
      • Zhou’s Blog
  • 最终解释权归 水怪 (Lufs) 所有 🤣

交换友链

如果你认为自己符合以上要求，那么就来交换啦～ 😆

添加本站～

  • 将本站添加至自己的友链页面中
    • 站点名称：Lufs's Blog
      • 非强制，合适即可
    • URL：https://isteed.cc/ 或 https://blog.isteed.cc
      • 俩个 URL 都可以的说，不过个人建议选第二个呢
    • 简介：一个假的博客~
      • 非强制，可随兴发挥～
    • Slogan：
      • Life feeds on negative entropy
      • 非强制，简介或 Slogan 自行选择～
    • Logo：
      • Favicon：
        • 16x16 png
        • 32x32 png
        • 180x180 png
        • 192x192 png
        • svg
      • Avatar：
        • 36x36 png
        • 72x72 png
        • 128x128 png
        • 256x256 png
        • 512x512 png
      • 自行选择合适的大小哈～

准备资料～

  • 准备一个自己想展示的站点图片（Logo）
    • 长度与宽度应小于 512px
    • 图片不宜过大（超过 1MB 的话，我会将其调整为 128px）
    • 使用 png、jpg、svg、ico 等常见文件格式哈～
      • webp Safari 不支持、icns 除了 Safari 外，其它基本都不支持
      • 为了兼容性考虑，请使用常见格式～
    • 使用链接形式（HTTPS ONLY!）
    • 尽量不要 NSFW（Not Safe/Suitable For Work）
  • 准备需要展示的友链名称
    • 尽量不要 NSFW
  • 准备一条 Slogan
    • 尽量不要 NSFW
